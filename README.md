# 🌦️ Weather & Air Quality Analysis

This project analyzes a global weather dataset (~136K records) to study the relationship between atmospheric conditions and air quality.

* Source:[Kaggle](https://www.kaggle.com/datasets/nelgiriyewithana/global-weather-repository?authuser=0) – Global Weather Repository
* Key focus: temperature, humidity, wind, and PM10 pollution
* Methods: data cleaning, EDA, visualisation, correlation analysis

*Insights:*

* PM10 increases with AQI → poorer air quality
* Temperature correlates strongly with feels-like temperature
* Most common condition: Partly Cloudy
